# Comment contribuer au projet
## Clone le projet
``` git clone https://github.com/PepinoFr/projet8-TodoList.git ```
## Créer une issue sur github
## Créer une branche avec le bouton sur l'issue
## Retourner sur le projet et taper ses commandes 
``` git fetch origin ```
``` git checkout 19-test ```
## Puis procéder aux modifications
## Lancer les tests
## Push les modifications et merge sur la branche main

#Task
## Pour ajouter des nouveaux champs allez dans entity/task
### Ajouter le nouveau champ en respectant la nomenclature
### Ajouter les getter et setter
### Ajouter le nouveau champ dans le formulaire src/form/TastType si besoin
### Ajouter le nouveau champ dans le test test/TastTest si besoin
### Ajouter le nouveau champ dans les dataFixtures si besoin
